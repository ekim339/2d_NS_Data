This repo contains ipynb notebook for generating 2D Navier Stokes data (vorticity, u velocity, v velocity) and a code for Physics Informed Diffusion Model (PIDM).

Code under directory PIDM is modified from the git repo below.
https://github.com/jhbastek/PhysicsInformedDiffusionModels

Link to paper: 
<a href="https://arxiv.org/abs/2403.14404">Physics Informed Diffusion Model (ICLR 2025)</a>
